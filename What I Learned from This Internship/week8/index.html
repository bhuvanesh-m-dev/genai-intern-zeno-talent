What I Learned

Tokenization & punctuation

Splitting text into tokens, removing punctuation marks like . , : ; ! ? using translate and word_tokenize.

Stop word removal

Filtering high‑frequency words (the, is, in) that carry little semantic weight in tasks like classification.

Stemming vs lemmatization

PorterStemmer chops suffixes (running→run), WordNetLemmatizer uses vocabulary (better→good).

preprocess.py
import string, nltk
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer, WordNetLemmatizer
text = para1 + para2
# remove punctuation
no_punct = text.translate(str.maketrans('', '', string.punctuation))
# tokenize, stopwords, stem, lemma
tokens = word_tokenize(no_punct.lower())
filtered = [w for w in tokens if w not in stop_words]
stemmed = [PorterStemmer().stem(w) for w in filtered]
lemmatized = [WordNetLemmatizer().lemmatize(w) for w in filtered]

1. Remove punctuation str.translate 
import string
text = para1 + " " + para2
text_no_punct = text.translate(
    str.maketrans('', '', string.punctuation)
)
# Output: 'Technology is evolving rapidly ...'
2. Stop word removal word_tokenize + stopwords.words() 
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

tokens = word_tokenize(text_no_punct)
stop_words = set(stopwords.words('english'))
filtered = [w for w in tokens 
            if w.lower() not in stop_words]
3. Porter Stemming PorterStemmer 
from nltk.stem import PorterStemmer
stemmer = PorterStemmer()
stemmed = [stemmer.stem(w) for w in filtered]
# 'evolving' → 'evolv', 'creativity' → 'creativ'
4. WordNet Lemmatization WordNetLemmatizer 
from nltk.stem import WordNetLemmatizer
lemmatizer = WordNetLemmatizer()
lemmatized = [lemmatizer.lemmatize(w) for w in filtered]
# 'machines' → 'machine', 'humans' → 'human'

Real‑world NLP pipelines

Where text cleaning is essential

Every NLP model — from sentiment analysis to chatbots — requires raw text to be normalized. Stopword removal reduces noise, lemmatization consolidates inflected forms, and punctuation removal simplifies token boundaries.

1

Sentiment Analysis

Clean tweets and reviews: remove mentions, punctuation, stopwords before feeding to classifier.

2

Information Retrieval

Search engines stem/lemmatize queries and documents to improve recall.

3

Chatbots / RAG

Preprocess user input to extract keywords and match intents after lemmatization.

Learning notes ↗

Typical pipeline

1. Raw text → lowercase
2. Remove digits/punctuation
3. Tokenize
4. Stopword removal
5. Stemming / lemmatization
6. → vector representation
Example: “running faster!” → [‘run’, ‘fast’]
