Mean Squared Error (MSE)

MSE penalizes larger errors more heavily because the errors are squared. This makes it particularly useful when we want to strongly discourage large deviations in predictions.

  • Squaring errors amplifies sensitivity to outliers.
  • Differentiable function, making it easier for gradient descent.

Mean Absolute Error (MAE)

MAE treats all errors uniformly and is generally more robust to outliers. It provides a more balanced view of overall error by measuring the absolute differences between predictions and actual values.

  • Linear penalty means outliers don't skew the metric excessively.
  • More intuitive interpretation of average error magnitude.

Root Mean Squared Error (RMSE)

Derived from MSE, RMSE is expressed in the same units as the original values, making it significantly easier to interpret in real-world scenarios while still retaining the property of penalizing large errors.

  • Same units as the target variable (e.g., dollars, meters).
  • Bridges the gap between interpretability and sensitivity to large mistakes.

Key Learnings

Sensitivity

Different loss functions react differently to small and large prediction errors. MSE increases rapidly due to squaring, making it highly sensitive to big errors.

Stability

MAE is more stable and less affected by extreme values or outliers, offering a "fair" assessment of general error.

Interpretability

RMSE helps interpret the error in original value units while still emphasizing larger mistakes, balancing utility and rigor.

Application

Visualizing loss functions is crucial for understanding model behavior, evaluating performance, and making optimization decisions.
