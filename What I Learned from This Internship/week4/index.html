Activation Functions

Exploring neural network non-linearity. How Sigmoid, Tanh, and ReLU transform inputs and influence model learning behavior.

Start Exploration View Code

Sigmoid Activation

Maps inputs into a range between 0 and 1, forming an S-shaped curve. Commonly used in binary classification models to predict probabilities.

  • Can cause vanishing-gradient issues for extreme values.
  • Smooth gradient, preventing jumps in output.

Tanh Activation

Outputs values between -1 and +1. It is zero-centered, which generally helps in gradient-based learning compared to Sigmoid.

  • Zero-centered output aids convergence.
  • Still susceptible to gradient saturation at extremes.

ReLU (Rectified Linear Unit)

Outputs zero for negative values and a linear response for positive values. It is computationally efficient and currently the most widely used activation function.

  • Fast computation and convergence.
  • Risk of "Dead Neurons" where neurons stop activating entirely.

Leaky ReLU

Introduces a small slope for negative values (e.g., 0.01x), preventing neurons from becoming inactive and ensuring gradient flow continues during training.

  • Solves the "Dead ReLU" problem.
  • Maintains advantages of ReLU while improving stability.

Key Takeaways

Non-Linearity

Activation functions introduce non-linearity, enabling neural networks to learn complex patterns beyond simple linear regression.

Gradient Flow

Different functions affect training speed. Sigmoid/Tanh can vanish, while ReLU families generally support faster training.

Efficiency vs. Dead Neurons

ReLU is computationally cheap but risks dead neurons. Leaky ReLU fixes this by allowing small negative gradients.
